# 부산지역 RISE 체계 내 외국인 유학생 교육체계 구축 연구

## 프로젝트 개요

이 리포지토리는 신진연구자지원사업(인문사회분야)을 위한 연구계획서를 포함하고 있습니다.

## 연구 주제

부산지역 RISE(Regional Innovation System and Education) 체계 내 외국인 유학생 교육체계 구축 연구

## 문서

- **[research_proposal.md](research_proposal.md)**: 신진연구자지원사업 연구계획서 전문

## 연구 목표

1. 부산지역 대학의 외국인 유학생 교육 현황 및 사례 분석
2. 산업체, 유학생, 대학관계자의 요구 및 인식 조사
3. RISE 체계에 적합한 외국인 유학생 교육모델 개발
4. 교육과정, 교육내용(교과+비교과), 학생지원 체계의 구체적 방안 제시

## 연구 기간 및 예산

- **기간**: 2년 (24개월)
- **예산**: 40,000,000원

## 예상 연구 성과

- KCI급 학술논문 3편 발표
- 부산지역 외국인 유학생 교육체계 모델 개발
- 정책 제언 및 실행 가이드북 발간

## 사용 방법

1. `research_proposal.md` 파일을 다운로드하여 검토합니다.
2. 연구책임자 정보를 기입합니다.
3. 소속 기관의 신진연구자지원사업 양식에 맞춰 필요한 부분을 수정합니다.
4. 신청 기관의 제출 지침에 따라 제출합니다.
